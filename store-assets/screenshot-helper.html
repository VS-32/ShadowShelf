📸 ShadowShelf Screenshot Helper

Opens each store screenshot at exactly 1280×800

Before you start

  1. Load the extension: chrome://extensions → Enable Developer Mode → Load unpacked → select dist/
  2. Use ShadowShelf for a few minutes so you have real data (visit a few sites, start a timer)
  3. For each card below, click the button — a new 1280×800 window opens
  4. Open DevTools in that window (F12) → Ctrl+Shift+P → type Capture screenshot → Enter
  5. Rename saved files: screenshot-1.png through screenshot-5.png
